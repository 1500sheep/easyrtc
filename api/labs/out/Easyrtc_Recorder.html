Easyrtc_Recorder

recordToFile(mediaStream, downloadLink, basename)

This method creates a media recorder that builds a file. Use the media recorder's start(), stop(), pause() and resume() methods on the returned object.
Parameters:
Name Type Description
mediaStream a local or remote media stream.
downloadLink Object an anchor tag to attach the file to.
basename String the name of the file. A .webm will be appended to the file if its not already present. The file doesn't get written until you call the mediaRecorder's stop method.
